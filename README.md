# Number-Guessing-Game

This is a simple console-based Number Guessing Game written in C.

Player 1 enters a secret number, and Player 2 has to guess it.
The program tells if the guess is too high, too low, or close to the correct number.

5. Counts total attempts



How It Works

Player 1 enters a number

Player 2 guesses again and again

The game gives hints and ends when the correct number is guessed
